# 🧙 Les Sœurs Potter — Site Fan Harry Potter

> Site web thématique dédié à l'univers magique de Harry Potter — Projet personnel réalisé en HTML/CSS/JavaScript.

---

## 📖 Description

Site encyclopédique et de présentation pour les passionnés de l'univers Harry Potter créé par J.K. Rowling. Le projet couvre plusieurs thématiques de la saga à travers une interface immersive et thématisée.

### Pages disponibles

| Page | Contenu |
|------|---------|
| 🏠 Accueil | Présentation du site et introduction à l'univers |
| 🧸 Collection | Objets et artefacts du monde magique |
| 🧙 Personnages | Présentation des personnages emblématiques |
| 🏰 Maisons | Les 4 maisons de Poudlard |
| ☠️ Horcruxes | Les horcruxes de Voldemort |
| 🎬 Films | La saga cinématographique |
| ✈️ Voyage | Les lieux iconiques de l'univers |
| 🎮 Jeu vidéo | L'univers Harry Potter dans le jeu vidéo |
